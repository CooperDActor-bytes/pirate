Thank you for purchasing Microsoft® Flight Simulator X: Acceleration. These notes, instructions, and tips will help you get the most out of the program.

IMPORTANT: This product is for entertainment purposes only and should not be used for training purposes. It is not part of an approved training program under the standards of the FAA or any other regulatory agency.

Minimum System Requirements

  • 2.0 GHz processor or higher.
  • 1.0 GB RAM
  • 128MB video RAM or higher.
  • Must have vertex and pixel shader model 1.1 or higher (no more hardware T&L).

Installing Flight Simulator X: Acceleration

If your computer does not support automatic installation, we recommend that you install Flight Simulator X: Acceleration per the following instructions to ensure the best experience.

  • You must have Flight Simulator X Standard or Deluxe installed and activated before installing Flight Simulator X: Acceleration.
  • If you have not installed Flight Simulator X Service Pack 1 (from fsinsider.com) before installing the expansion pack, Flight Simulator X: Acceleration Setup will install it.

Installation Notes

  • If you uninstall Flight Simulator X: Acceleration, your system reverts to the configuration it was in prior to installing the expansion pack.
  • If you install a version of Flight Simulator X: Acceleration that is in a different language than the language you use on your operating system, you may see a mixture of languages in the user-interface. If you do not have the fonts for the non-standard language, words appear unreadable.

To install on Windows® XP

  1. On the Start menu, click Control Panel.
  2. In the Category View, under Pick a Category, click Add or Remove Programs.
  3. Under Pick a Task, click Add a Program.
  4. In the Add or Remove Programs dialog box, click CD or Floppy, and then follow the on-screen instructions.

To install on Windows Vista®

  1. On the Start menu, click Computer.
  2. Under Devices with Removable Storage, double-click the disc drive icon.
  3. In the User Account Control dialog box, click Allow, and then follow the on-screen instructions.

NOTE: If you install Flight Simulator X: Acceleration on a computer running the 64-bit Vista operating system, the Setup progress bar may not show progress during the installation. The program still installs correctly on this operating system.

Starting Flight Simulator X: Acceleration



To start on Windows XP

  1. On the Start menu, point to Programs, point to Microsoft Games, and then point to Microsoft Flight Simulator X.
  2. Click Microsoft Flight Simulator X.

To start on Windows Vista

  1. Click Start, click Games, and then click Games Explorer.
  2. Double-click Microsoft Flight Simulator X: Acceleration.

Installation Tips
If you have problems installing Flight Simulator X: Acceleration, close all open software applications including antivirus software before running Setup.

Microsoft Flight Simulator X and Microsoft Windows Vista

Flight Simulator X: Acceleration is compatible with the Windows Vista operating system.

  • If you start the Input Method Editor (IME) and then start a Flight Simulator X multiplayer session, the IME language bar may be permanently hidden when you quit Flight Simulator X. You may have to restart Windows Vista to see the IME again.
  • If you install Flight Simulator X: Acceleration on Windows XP and then upgrade to Windows Vista, you see an activation error when you start Flight Simulator X. You must re-activate Flight Simulator X, reinstall the expansion pack, and then re-activate the expansion pack.
  • When running Flight Simulator X: Acceleration in windowed mode (as opposed to full screen mode) on Windows Vista, the program may appear to stop operating. Flight Simulator X: Acceleration continues running but you may want to temporarily disable the Windows Vista Glass and Aero features and refresh the Flight Simulator X: Acceleration window (see the Windows Vista Updates section below as well).

To disable Windows Vista features globally

  1. Right-click the Windows desktop and then click Personalize.
  2. Click Windows Color and Appearance.
  3. Clear the Enable Transparency check box.
  4. In the color scheme list, select Windows Vista Basic.
  5. Click OK.

To enable Windows Vista features globally

  1. Right-click the Windows desktop and then click Personalize.
  2. Click Windows Color and Appearance.
  3. Select the Enable Transparency check box.
  4. In the color scheme list, select Windows Vista Aero.
  5. Click OK.

To disable Vista features only when running Flight Simulator

  1. Open Windows Explorer and navigate to the folder in which you installed Flight Simulator X.
  2. Right-click FSX.exe and then click Properties.
  3. On the Compatibility tab, select the Disable desktop composition check box.
  4. Click OK.

Hardware Issues

For all hardware issues, always begin troubleshooting by downloading and installing the latest device drivers from the manufacturer's Web site.

Toshiba Qosmio laptop, Vista OS, GeForce 8600m GT display adapter
When you start Flight Simulator X: Acceleration on a computer with this configuration, you may get a message reading, "Minimum requirements not met." The error message is incorrect, as this configuration does meet the program's minimum requirements. You should, however, install the latest display drivers for this adapter and try running Flight Simulator again.

Product Activation

You must install and activate Flight Simulator X before installing Flight Simulator X: Acceleration. Once you install Flight Simulator X: Acceleration, you must activate your copy of the expansion pack. You can activate the product over the Internet using the Activation Wizard at the end of the installation. The Activation Wizard will appear every time you start Flight Simulator X: Acceleration until you activate the program. Until you activate the program, you can only use the product for 30 minutes each session.

The phone number for Korea does not appear correctly in the Activation Wizard user interface. The Korean product activation number is:

TollFree=1577-9700

DirectX

If you encounter any DirectX® errors after installing Flight Simulator X: Acceleration, you can manually download and install DirectX from microsoft.com/directx.

If you do not have an Internet connection, you can install DirectX from your hard drive:

  1. Navigate to the folder where you installed Flight Simulator X: Acceleration. The default location is c:\Program Files\Microsoft Games\\Microsoft Flight Simulator X\Setup\DX.
  2. Double-click DXSETUP.exe.

Other Issues

Sound Settings
We modified the Sound Settings dialog box so you can select separate audio adapters directly in the Sound Settings dialog box. We removed the Configure button and provided two list boxes to select audio adapters to separate voice and other sounds.

Multiplayer and Service Pack 1
After installing the Flight Simulator X Service Pack 1 and launching a multiplayer session, if you get an error that reads "An invalid argument was encountered", uninstall Service Pack 1 and reinstall it.

Software Development Kit Service Packs
You must have the Flight Simulator X SDK installed before installing the Flight Simulator X: Acceleration SDK. When installing the Acceleration SDK, if you get a message that reads "You must have SDK SP1a or SDK SP2," you can find the SDK SP1a update in the \SDK\SP1a folder on the Acceleration disk, or you can download and install the SDK service packs from fsinsider.com. Then try installing the Flight Simulator X: Acceleration SDK again.

  • Download SDK SP1a
  • Download SDK SP2 when available

Russian Software Development Kit Files
The Russian-language version of the Flight Simulator X Service Pack 1 update has some file incompatibilities with the English-only release of the SDK SP1a update. These issues will be addressed in the Flight Simulator X Service Pack 2 update of both Flight Simulator X and the Flight Simulator X SDK, which will become available on fsinsider.com.

The following SDK Service Pack 1a tools are not compatible with the Flight Simulator X Service Pack 1 Russian runtime:

  • Object_placement.dll
  • Visual_fx.dll
  • Traffictoolbox.dll

The installation of the SimConnect.msi as provided in the SDK Service Pack 1a is not compatible with the Flight Simulator X Service Pack 1 Russian runtime:

  • SimConnect client applications that are run on a remote machine (i.e., connecting to Flight Simulator X Service Pack 1 RUS on a different machine) will not connect properly.
  • This problem does not affect client applications run on the same machine as Flight Simulator X Service Pack 1 RUS.
